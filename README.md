# POC DynamoDB


## GetStarted

- Before, run the environments
  - This compose run the local dynamoDB in  _localhost:8000_ and the admin interface in _localhost:8001_

```bash
docker-compose up -d --build
``` 

- To run the program 

```bash
yarn start:dev
```
